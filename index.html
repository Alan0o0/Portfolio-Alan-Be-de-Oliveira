Sobre o projeto

Consulta-Cep é uma aplicação desenvolvida apenas em Java , que realiza consultas de CEP de forma rápida e eficiente, retornando informações como rua, bairro, cidade e estado. O projeto demonstra conhecimentos em POO, requisições HTTP e consumo de APIs, oferecendo uma solução prática para automatizar consultas de endereço.
